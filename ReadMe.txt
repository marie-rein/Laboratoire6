Dans le package de Nuget j'ai trouvé une fonction appelée WriteLineFormatted()
Cette fonction te permet de formatter un texte avec differentes couleurs dans une meme ligne

Console.WriteLineFormatted(monString, Color.OrangeRed, Color.White, composantsOrdinateur);

Dans cet example j'ai deux textes, monString et composantsOrdinateur, donc dans cette ligne de code je met la couleur sur monString et sur composantOrdinateur mais c'est deux couleurs differentes
on peux meme mixer les textes avec ces differentes couleurs et avoir une phrase avec deux couleurs.
